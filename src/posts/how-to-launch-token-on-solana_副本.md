# 如何在 Solana 上发行代币

Solana 是一个高性能的区块链平台，非常适合发行代币。本文将指导您完成在 Solana 上发行代币的步骤。

## 准备工作

1. 安装 Solana CLI
2. 创建 Solana 钱包

## 创建代币

使用以下命令创建代币：

```bash
solana-token create-token
```

## 发行代币

使用以下命令发行代币：

```bash
solana-token create-token
```

## 验证代币

使用以下命令验证代币：

```bash
solana-token verify-token
```
